Detailed Description

Directory parsing functions.

Author:
albinoloverats ~ Software Development
Date:
2009-2010
Note:
These functions all return newly allocated strings

Collection of functions to parse directory hierarchies and extract the names of each leaf along the tree, from the root all the way to the file name and password
